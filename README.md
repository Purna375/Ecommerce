# Male Fashion E-Commerce Website

## Description
This is a front-end e-commerce website designed for male fashion products. It features a responsive design with various pages including home, shop, product details, shopping cart, checkout, blog, and contact sections. The site is built using static HTML, CSS, and JavaScript, with SASS for styling.

## Features
- Responsive design for mobile and desktop
- Product catalog with categories
- Product detail pages
- Shopping cart functionality
- Checkout process
- Blog section
- Contact and FAQ pages
- User sign-in page

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- SASS/SCSS
- Bootstrap
- jQuery
- Owl Carousel
- Magnific Popup
- SlickNav
- MixItUp

## Installation
1. Clone or download the repository to your local machine.
2. Ensure you have a web server or use a local development server to serve the files.

## Usage
1. Open `index.html` in your web browser to view the home page.
2. Navigate through the site using the menu links.
3. For development, compile SASS files if needed (style.scss to style.css).

## File Structure
- `index.html` - Home page
- `shop.html` - Shop page
- `shop-details.html` - Product details
- `shopping-cart.html` - Cart page
- `checkout.html` - Checkout page
- `blog.html` - Blog listing
- `blog-details.html` - Blog post details
- `contact.html` - Contact page
- `faq.html` - FAQ page
- `signin.html` - Sign-in page
- `about.html` - About page
- `css/` - Compiled CSS files
- `js/` - JavaScript files
- `sass/` - SASS source files
- `img/` - Images and assets
- `fonts/` - Font files

## Development
- Edit SASS files in the `sass/` directory and compile to `css/style.css`.
- Modify HTML files directly.
- Update JavaScript in the `js/` directory.
